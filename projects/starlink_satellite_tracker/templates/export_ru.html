Доступ к API

Программный доступ к данным о спутниках через наш RESTful API:

Конечные точки API
  • GET /api/satellites - Получить текущие позиции спутников
  • GET /api/passes?lat=:lat&lon=:lon&hours=:hours - Получить прогнозируемые пролеты
  • GET /api/coverage - Получить данные о глобальном покрытии
  • GET /api/export/json - Экспорт данных в формате JSON
  • GET /api/export/csv - Экспорт данных в формате CSV
